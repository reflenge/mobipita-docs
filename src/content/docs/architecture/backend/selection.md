## 概要

MobiPitaでは、リアルタイム機能やサーバーレス関数を提供するバックエンドサービスが必要です。

以下のバックエンドサービスを検討しています。

## バックエンドサービス候補の比較

### Convex（検討中）

**特徴:**
- リアルタイムデータベースとサーバーレス関数
- TypeScriptファースト
- リアルタイム同期機能

**メリット:**
- リアルタイム機能が標準で提供されている
- TypeScriptファーストで開発体験が良い
- サーバーレス関数とデータベースが統合されている
- 開発速度が速い

**デメリット:**
- 比較的新しいサービスでエコシステムが小さい
- ベンダーロックインのリスク
- コストが高くなる可能性

**マルチテナント対応:**
- テナントごとのデータ分離を実装する必要がある
- 関数レベルでテナント分離を実装

### その他の選択肢

**Next.js API Routes / Server Actions**
- Next.jsに統合されたサーバーレス関数
- 追加のサービスが不要
- リアルタイム機能は別途実装が必要

**AWS Lambda + API Gateway**
- サーバーレス関数の実行
- 高いスケーラビリティ
- リアルタイム機能は別途実装が必要

## 推奨選定

**検討中**: Convex

**選定基準:**
- リアルタイム機能が必要な場合: Convex
- シンプルなAPIのみ必要な場合: Next.js API Routes / Server Actions

詳細は実装フェーズで決定予定です。
