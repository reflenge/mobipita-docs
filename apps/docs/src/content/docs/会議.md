## master2B2B2C 構成 について

$master \to B_1$ と $B_1 \to B_2 \to User$ を分ける。

```mermaid

    subgraph "Next.js(Master App)"
        master[マスター管理画面]

        CompanyA --> ShopsA1
        CompanyA --> ShopsA2
        CompanyA --> ShopsA3

        ShopsA2 <--> User1

    end
```

## Customer App

### App

`Vercel` or `AWS`
マルチテナントサービスをホストするための環境があるらしい

サブドメイン環境を使う

https://<会社>.mobipita.com

### Auth
`Clerk` or `Better Auth`

会社ごとにそれぞれ用意する

### DB
`Cloudflare D1` or `AWS RDS ?`
会社ごとにそれぞれを用意する

### Storage
`AWS S3` or `Cloudflare R2`

会社ごとにそれぞれを用意する

料金による

### Payment
`Stripe` or `Polar`

統一環境?
